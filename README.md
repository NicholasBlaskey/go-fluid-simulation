### Building

Install needed deps
```
go get -u github.com/go-gl/mathgl/mgl32
go get -u github.com/go-gl/gl/v4.1-core/gl
go get -u github.com/go-gl/glfw/v3.3/glfw
```

Then run
```
go run main.go
```
